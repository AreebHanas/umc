# Screenshots Folder

This folder contains screenshots of the application for documentation purposes.

## Required Screenshots

Please add the following screenshots:

1. **home.png** - Main landing page with header and footer
2. **dashboard.png** - User dashboard with sidebar navigation
3. **auth.png** - Login/Register authentication page
4. **admin.png** - Admin panel interface

## Screenshot Guidelines

- Use high-quality images (PNG format recommended)
- Capture full browser window or relevant sections
- Use consistent browser window size (1920x1080 recommended)
- Include navigation and key features
- Remove any sensitive information

## How to Add Screenshots

1. Take screenshots of your running application
2. Save them with the names mentioned above
3. Place them in this folder
4. Screenshots will automatically appear in the GitHub.md file
